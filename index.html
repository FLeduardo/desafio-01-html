o que é HTML

HTML,que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto, em português) é a linguagem de marcação padrão usada para criar páginas da web.
Foi desenvolvida para descrever a estrutura e o conteúdo de uma página da web, permitindo que os navegadores da web interpretem e exibam essa página de forma consistente para os usuários.

Como funciona HTML

O HTML ,funciona como uma linguagem de marcação usada para criar documentos hipertexto, como páginas da web. Sua função principal é estruturar o conteúdo de uma página da web, indicando ao navegador. como deve ser exibido e organizado o texto, imagens, links e outros elementos.

Falando sobre Tags

Tags são elementos fundamentais em linguagens de marcação, como o HTML,que são usadas para definir a estrutura e a formatação de um documento. Elas são como comandos ou rótulos que indicam ao navegador da web como interpretar e exibir o conteúdo dentro de um documento. As tags são inseridas dentro do código-fonte de um documento e são delimitadas por colchetes angulares ("<" e ">").

As tags têm duas partes principais:

Tag de Abertura: A primeira parte de uma tag é a tag de abertura, que começa com "<" seguido pelo nome da tag e, opcionalmente, atributos.
